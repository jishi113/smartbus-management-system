# 🚌 SmartBus – Bus Management System

SmartBus is a full-stack Bus Management System built using React and Spring Boot.  
The application allows users to manage bus details including routes, source, destination, and bus numbers.

---

## 🚀 Features

- Add new buses
- View all bus details
- Update bus information
- Delete buses
- Responsive UI using Bootstrap
- REST API integration

---

## ⚙️ How to Run the Project

### Backend


cd springboot-backend
./mvnw spring-boot:run


### Frontend


cd react-frontend
npm install
npm start
